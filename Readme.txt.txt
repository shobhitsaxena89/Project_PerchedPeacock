PerchedPeacock Parking
------------------------------


System Requirements

Processor : Any (x86 | x64)
Windows : Win 7/8/10
Windows Server: Windows Server 12/16/19
Server Memory : 4GB
Client Memory : 2GB
Graphics Card : NA


System Installations:

On Server:


IIS 6.0 for deployment
Microsoft .Net Framework 4.7.2 or above

Db Installation:

Sqlite3 for storage/Database
sqlite Studio for Db tool

Node Installation :

1. Download and Install Node js Setup from (https://nodejs.org/en/download/)
2. Verify that Node.js was Properly installed,
	open cmd -> node -v (it should tell the version of node installed).
3. Update the Local npm Version
	npm install npm --global

Deploy the package on Server and start surfing PerchedPeacock
